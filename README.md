# Forward IPTV 模块

这是一个用于在 Forward 播放器中观看 M3U IPTV 直播的模块，支持频道分组、EPG 节目单、以及同组频道快速切换（支持“频道收藏”机制）。

## 安装与使用

1. 打开 Forward 播放器。
2. 导入当前目录下的 `iptv.js` 文件。
3. 在模块设置中，配置你的 M3U 订阅链接和 EPG 节目单链接。
   - 默认 M3U 订阅链接为：`/Users/skyland/Downloads/Emby IPTV (1).m3u`（如果是手机端测试，由于无法直接读取电脑本地文件，你需要将此文件放到可访问的网络路径下，或者在 iPhone 本地路径）。
   - 默认 EPG 节目单链接为：`http://epg.51zmt.top:8000/e.xml`。

## 功能特性
- **UI 统一**：解析 M3U 中的分组（`group-title`），以类似剧集分类的形式展示。
- **节目单支持**：自动抓取 EPG 并通过匹配 `tvg-id` 或 `tvg-name` 在详情页显示频道节目表，正在播出的节目会有“👈 正在播出”标记。
- **快速切换和收藏**：将同一分组下的所有频道包装为 `episodeItems`（选集），你在播放页面可以通过选集列表一键切换其他频道。同时 Forward 原生支持将此类 `VideoItem` 进行收藏。
